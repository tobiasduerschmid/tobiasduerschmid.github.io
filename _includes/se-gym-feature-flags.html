{%- comment -%} Emits window.SEGymFeatures from _config.yml's `se_gym_features` map so the SE Gym JS can turn individual features on/off. A missing config key defaults to ON; only an explicit `false` disables a feature. Any value already present on window.SEGymFeatures (e.g. set by a test via page.addInitScript before page scripts run) wins over config, so individual features can be exercised in tests regardless of the build's defaults. Include this on every page that runs SE Gym JS, before that JS executes. {%- endcomment -%}
