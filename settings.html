My Settings

These are the preferences this site lets you change. They are stored only in your browser as cookies or localStorage; nothing is sent to a server.

To inspect or delete raw stored values, use Browser storage.

Appearance and motion

Dark mode

Use the site's dark color theme.

Cookie: dark-mode

Text highlights

Show highlighted passages in articles and SEBook pages.

Cookie: show-highlights

Motion preference

Follow your operating system's reduced-motion preference, or always reduce optional animations on this site.

localStorage: prefersReducedMotion

Glossary abbreviation underlines

Show visible dotted underlines on glossary abbreviations in body content. Headings never receive automatic abbreviation markup.

localStorage: abbr-underlines
Reading and audio

Read-aloud controls

Show text-to-speech controls on supported pages.

Cookie: read-aloud

Read-aloud voice

Choose the browser voice used by the read-aloud feature.

localStorage: tts-voice-name

Voice choices depend on your browser and operating system.

Audio volume

Default volume for embedded audio players.

localStorage: cap_volume

Audio speed

Default playback speed for embedded audio players.

localStorage: cap_speed
Tutorials and diagrams

Tutorial autosave

Automatically save tutorial code and current step in this browser.

localStorage: tutorial-autosave

UML accent color

Custom accent color used by supported UML diagram renderers.

Cookie: uml-accent-color
Personal study tools

SEBook bookmarks

Show bookmark controls on SEBook pages.

Cookie: se-bookmarks-active

SE Gym

Enable the personal question and flashcard gym.

Cookie: se-gym-active

Performance tracking

Track seen/correct counts locally so SE Gym can surface your most difficult questions.

Cookie: analyze-performance; localStorage: se-gym-stats
Per-page tutorial state

Tutorial editor layout

Individual tutorials remember editor split sizes, popout recovery state, debugger breakpoints, and collapsed debugger panels on the page where you change them.

localStorage prefixes: tutorial-editor-split-, tutorial-popout-state-, tutorial-debug-bps-, tutorial-debug-watchpoint-remove-choice-, tutorial-debug-section-, tutorial-debug-subsection-

Change these inside each tutorial. Delete stored values from Browser storage.

Tutorial progress and code

When autosave is enabled, tutorial code, current step, and standalone tutorial progress are saved per tutorial.

localStorage prefixes/keys: tutorial-progress-, regex-tutorial-progress, regex-tutorial-advanced-progress

Use Tutorial autosave above to choose whether new tutorial work is saved.
