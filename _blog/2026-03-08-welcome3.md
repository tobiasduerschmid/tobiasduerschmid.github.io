## How to Add Blog Posts 2

To add a new blog post, simply create a new markdown file (`.md`) in the `/blog` folder with the following format:

```
---
layout: blog-post
title: "Your Post Title"
date: 2026-03-09
category: "Your Category"
---

Your post content goes here in markdown format.
```

### Front Matter Fields

- **layout**: Always use `blog-post`
- **title**: The title of your post
- **date**: The publication date in `YYYY-MM-DD` format
- **category**: The category for filtering (optional, but recommended)

## Features

- Posts are automatically sorted by date (newest first)
- Posts can be filtered by category
- Each post gets its own URL at `/blog/your-post-slug/`
- The blog index page shows all posts with excerpts

## Example Categories

You can use any category you like. Here are some suggestions:

- Teaching
- Research
- General
- Software Engineering
- Computer Science Education

Just drop your markdown files in the `/blog` folder and Jekyll will handle the rest!
